给博客添加自定义域名

因为之前跟风给自己弄了一个域名邮箱，既然都有域名了，就还是搞一下吧。 DNS设置 本身在搞icloud域名邮箱的时候，就已经添加了好几条了。但都是关于邮件收发的，大概是不冲突的（？ CNAME： 名称www，内容为user.github.io 根据github文档，至少得添加一些A记录，选@就行： 185.199.108.153 185.199.109.153 185.199.110.153 185.199.111.153 代理不用选，直接github也可以给https Github Pages设置 改博客的config文件，将baseURL改为"https://www.example.com/" 在static文件夹里创建CNAME，里面写上www.example.com 因为是用git action进行更新的，每次push都会重新生成，所以要放在static，保证它不会每次都被影响。因为在上面dns我是设了www，所以感觉这里还是写www吧 push之后，到github pages的setting - pages，将Enforce HTTPS打勾 永久链接 以此为契机，也重新把url设置了一下 博客config文件里加上permalinks（和params同级） permalinks: posts: "/posts/:slug/" 而关于slug，则参考Hugo博客永久链接 · 整点内啡肽 (happytoo.cyou)进行了设置 slug: {{ substr (md5 (printf "%s%s" .Date (replace .TranslationBaseName "-" " " | title))) 4 8 }}
