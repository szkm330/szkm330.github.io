v2ray自用流程

2019-06-03 · szkm330

用的是Ubuntu16.04，主要流程如下：

# 刷新源及更新软件
sudo apt-get update && sudo apt-get upgrade
# 安装 curl
sudo apt install curl
如果不是 root 环境需要进入
sudo su
接着就可以安装 Project V(V2Ray) 了
--以上：不知道是不是必须，姑且做一下--   
       
bash <(curl -L -s https://install.direct/go.sh)
# 上列命令会自动安装 V2Ray，然后执行下面命令运行
systemctl start v2ray

记下端口号和id　　

vim /etc/v2ray/config.json进行修改和查看 systemctl start|restart|stop|status v2ray其余控制 https://www.uuidgenerator.net/ 可以生成ID

客户端相关 下载v2ray-windows-64.zip（然后直接编辑config似乎就可以了） 不过可以继续下载v2rayN.zip解压至同一个路径下 接着把相应信息填进去，大概就可以了

若要升级，直接再运行一次安装脚本
