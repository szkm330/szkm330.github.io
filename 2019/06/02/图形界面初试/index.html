图形界面初试

发表于 2019-06-02 | 分类于 python

首先画图，保存test.ui。
进入test.ui所在文件目录，输入命令pyuic5 test.ui –o test.py
将test.py开头的from PyQt5 import这行抄进去
再加上import sys import test

有两种基类QMainWindow和QWidget，使用时可以用一下代码。

# QMainWindow的场合
if __name__ =='__main__':
    app = QtWidgets.QApplication(sys.argv)
    MainWindow = QtWidgets.QMainWindow()
    ui = test.Ui_MainWindow()  # 照抄test.py里的类名
    ui.setupUi(MainWindow)
    MainWindow.show()
    sys.exit(app.exec_())


 # QWidget的场合
    if  __name__=="__main__":
        app=QtWidgets.QApplication(sys.argv)  
        widget=QtWidgets.QWidget()  
        ui=test.Ui_Form()  
        ui.setupUi(widget)  
        widget.show()  
        sys.exit(app.exec_())
