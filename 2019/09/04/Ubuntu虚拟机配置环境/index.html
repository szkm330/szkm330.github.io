Ubuntu虚拟机配置环境

发表于 2019-09-04 | 分类于 杂记

翻开课本，发现了 Linux的内容，便打算先行搞个虚拟机玩一下。
以下是大致流程。

1.下载
VirtualBox、Ubuntu18.04，没啥好说的，看网速。
新建虚拟机，选择合适参数。开机放碟装系统。
若是卡在”安装语言包”，可断网跳过，待换源后再下载。

2.换源

# 建立备份
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
# 编辑，之前先apt install一个vim
sudo vim /etc/apt/sources.list

使用清华源https://mirrors.tuna.tsinghua.edu.cn/help/ubuntu/

3.换源后的更新

sudo apt update
sudo apt upgrade

4.anaconda安装
下载完，再bash xxxxx.sh
一路回车与yes

# 改环境变量
sudo vim /etc/profile
# 最后加
export PATH=~/anaconda3/bin:$PATH
# 生效
source /etc/profile

5.vim配置

sudo spt install git
mkdir -p ~/.vim/bundle
git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim

配置完 :PluginInstall
删除插件 :PluginClean

附上完整配置，有待改进：

func! CompileRunGcc()
    exec "w"
    if &filetype == 'c'
        exec "!g++ % -o %<"
        exec "!time ./%<"
    elseif &filetype == 'cpp'
        exec "!g++ % -o %<"
        exec "!time ./%<"
    elseif &filetype == 'java'
        exec "!javac %"
        exec "!time java %<"
    elseif &filetype == 'sh'
        :!time bash %
    elseif &filetype == 'python'
        exec "!time python3 %"
    elseif &filetype == 'html'
        exec "!firefox % &"
    elseif &filetype == 'go'
        exec "!go build %<"
        exec "!time go run %"
    elseif &filetype == 'mkd'
        exec "!~/.vim/markdown.pl % > %.html &"
        exec "!firefox %.html &"
    endif
endfunc
