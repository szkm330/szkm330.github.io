v2ray坐井观天版

发表于 2019-08-22 | 分类于 杂记

我又来了，为了证书操碎了心。

1：基本安装

# 刷新源及更新软件
sudo apt-get update && sudo apt-get upgrade
# 脚本安装v2ray
bash <(curl -L -s https://install.direct/go.sh)
# 编辑 V2Ray 配置
vim /etc/v2ray/config.json

2：bbr

wget –no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh    
 chmod +x bbr.sh    
./bbr.sh

3：注册域名，A 记录指向自己

4：安装nginx

ubuntu安装nginx
sudo apt-get install nginx
开启nginx
sudo /etc/init.d/nginx start
开机自启动
systemctl enable nginx
重载nginx
/etc/init.d/nginx reload

5：安装certbot
依次

sudo apt-get update
sudo apt-get install software-properties-common
sudo add-apt-repository ppa:certbot/certbot
sudo apt-get update
sudo apt-get install python-certbot-nginx

执行certbot --nginx按指示选1、2，自动设置https
更新certbot renew # 关闭nginx
它只会更新快要过期的证书，所以频繁运行也没有问题
更新前可以用certbot renew –dry-run进行演习（？）
查看证书certbot certificates

6：检查
测试证书是否正常https://www.ssllabs.com/ssltest/index.html
